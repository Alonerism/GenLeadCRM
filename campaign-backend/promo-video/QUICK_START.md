## Keyboard Shortcuts in Remotion Studio

- **Space** - Play/Pause
- **Left/Right Arrow** - Step forward/backward by 1 frame
- **Shift + Left/Right** - Step by 10 frames
- **Home** - Jump to start
- **End** - Jump to end
- **L** - Loop on/off

## Need Help?

If the dropdown isn't showing up:
1. Make sure Remotion Studio loaded completely
2. Refresh the browser page
3. Check the terminal for any errors
4. Try stopping (`Ctrl+C`) and running `npm start` again

**Still can't find it?** The composition selector is typically in the header bar, near the Remotion logo.
